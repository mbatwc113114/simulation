Our Vision

The Simulation Project aims to create an open-source platform for developing and sharing interactive simulations. We strive to make technology more accessible, helping students, researchers, and enthusiasts build and experiment with real-world models.

Get Started

Clone the Repository:
git clone https://github.com/mbatwc113114/simulation.git

cd simulation

Install Dependencies:
pip install -r requirements.txt

Run the Application:
python main.py
